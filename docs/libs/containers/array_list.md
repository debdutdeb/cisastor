# Array List

`array_list` is a dynamic, growable array implementation for C. It provides a flexible way to store untyped data while offering a type-safe interface through macros.

## Why Array List?

In standard C, managing dynamic arrays requires manual tracking of size and capacity, along with frequent calls to `realloc`. This is error-prone and leads to repetitive code. `array_list` solves this by:
- **Automatic Resizing**: It grows its underlying memory region as you append items.
- **Encapsulation**: It tracks length, capacity, and element size internally.
- **Type Safety**: Through the `array_list_init` macro, it generates a custom, type-safe API for any struct or primitive type, preventing pointer-type mismatches.

## Usage

### Type-Safe Initialization
To use `array_list` with a specific type, you must first initialize it. This generates inline wrapper functions specific to that type.

```c
#include "containers/array_list.h"

typedef struct point { int x, y; } point;
// Generates functions like array_list_create_point, array_list_append_point, etc.
array_list_init(point);
```

### Basic Operations
```c
void example() {
    // 1. Create the list
    struct array_list_point *points = array_list_create_point();

    // 2. Append elements (passed by value)
    point p = {10, 20};
    array_list_append_point(points, p);

    // 3. Access elements (returns a pointer to the internal memory)
    point *p_ref = array_list_get_point_at(points, 0);
    if (p_ref) {
        printf("Point: %d, %d\n", p_ref->x, p_ref->y);
    }

    // 4. Check length
    size_t count = array_list_get_length((struct array_list *)points);
}
```

## API Reference

### Core Functions

| Function | Description |
| --- | --- |
| `array_list_create(size_t elem_size)` | Creates a generic list for elements of `elem_size`. |
| `array_list_append(list, const void *data)` | Appends data to the list. Returns a pointer to the new element. |
| `array_list_get_element_at(list, int index)` | Returns a pointer to the element at `index`, or `NULL` if out of bounds. |
| `array_list_get_length(list)` | Returns the current number of elements. |
| `array_list_get_capacity(list)` | Returns the total capacity before the next reallocation. |

### Generated Type-Safe API
Generated by `array_list_init(T)`:
- `array_list_create_T()`: Create a list for type `T`.
- `array_list_append_T(list, val)`: Append a value of type `T`.
- `array_list_get_T_at(list, index)`: Get a pointer of type `T*`.
- `iterator_element_T(it)`: Helper for iterator integration.
